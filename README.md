# Cuvette Micro Project Three: Javascript

### Overview

This repository contains the code for Cuvette Micro Project Three, The project focuses on implementing map, filter and sort method for array manipulation.

### Getting Started

Clone the repository to your local machine:

```bash
git clone https://github.com/chiranjitmu/cuvette-micro-project3.git
```

Open the index.js file and run through console.
